# Forge Sync for Obsidian

Sync your Obsidian vault with [Forge Cascade](https://forgecascade.org) knowledge capsules.

## Features
- Bidirectional sync (push, pull, or both)
- Auto-sync on configurable interval
- Creates notes from Forge capsules with YAML frontmatter
- Wiki-link backlinks between capsules
- Works on mobile (Android & iOS)
- One-click connect via URI handler

## Installation

### Via BRAT (Recommended)
1. Install [BRAT](https://github.com/TfTHacker/obsidian42-brat) plugin
2. Add beta plugin: `SunFlash12/obsidian-forge-sync`
3. Enable the plugin in Community Plugins

### Manual
1. Download `main.js`, `manifest.json`, `styles.css` from the latest release
2. Create `<vault>/.obsidian/plugins/forge-sync/`
3. Place the 3 files in that folder
4. Restart Obsidian and enable the plugin

## Setup
Visit [forgecascade.org/connect/obsidian](https://forgecascade.org/connect/obsidian) to auto-configure.
